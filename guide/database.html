Select Data

Select all data from a table.

        $this->database->from('my_table')->get();
        
        $this->database->get('my_table');

Select some data from a table.

        $this->database->select('column1, column2')->from('my_table')->get();

Setting Limits

        $this->database->get('my_table', $offset, $limit);

Filtering Data

Where

Select some data from a table using where clause.

        $this->database->from('my_table')->where('column1', 'value1')->get();

Custom operand.

        $this->database->from('my_table')->where('column1', 'value1', '>')->get();

Or Where

        $this->database->from('my_table')->where('column1', 'value1')->or_where('column2', 'value2')->get();

Where In

        $this->database->from('my_table')->where_in('column1', array('value1', 'value2'))->get();

Or Where In

        $this->database->from('my_table')->where_in('column1', array('value1', 'value2'))->or_where_in('column2', array('value3', 'value4'))->get();

Joining Tables

        $this->database->from('my_table')->join('second_table', 'mytable.id = second_table.id2')->get();

To use 'left join' and other they must be passed as third argument in the join function.

Avaiable options: "INNER", "LEFT", "RIGHT", "LEFT OUTER", "RIGHT OUTER".

        $this->database->from('my_table')->join('second_table', 'mytable.id = second_table.id2', 'LEFT')->get();
