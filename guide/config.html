Config

The config class allows the developer to load variable from config files to be used in the aplication.

It is a core class so you don't need to load it manually.

Loading Files

The configuration files are located in the config folder in the root path.

The folowing example will load the file config/myfile.php

$this->config->load('myfile');

Beware: config variables are rewritten if they have the same name.

Getting Values

To get a value from the config you need to use the get method.

$this->config->get('myvar');

Setting Values

These values are temporal. They will not be stored in any way, just replace the actual value.

$this->config->set('myvar', 'myvalue');
